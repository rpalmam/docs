## 1. Get your API key

Purchase a module at [enrichlatam.com](https://enrichlatam.com). Your API key will be sent to your email immediately after checkout.

Your key looks like this:

```
sk_a1b2c3d4e5f6...
```

## 2. Make your first request

Use your key in the `X-API-Key` header. Here's a KYB verification for a Chilean company by RUT:

<CodeGroup>

```bash cURL
curl -X POST https://api.enrichlatam.com/v1/kyb \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"identifier": "76354771-K", "country": "CL"}'
```

```python Python
import requests

response = requests.post(
    "https://api.enrichlatam.com/v1/kyb",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={"identifier": "76354771-K", "country": "CL"}
)
print(response.json())
```

```javascript Node.js
const response = await fetch("https://api.enrichlatam.com/v1/kyb", {
  method: "POST",
  headers: {
    "X-API-Key": "YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ identifier: "76354771-K", country: "CL" })
});
const data = await response.json();
console.log(data);
```

</CodeGroup>

## 3. Read the response

```json
{
  "identifier": "76354771-K",
  "country": "CL",
  "company_name": "Empresa Ejemplo Spa",
  "status": "active",
  "verified": true,
  "founded_year": 2018,
  "end_date": null,
  "source": "sii"
}
```

<Check>`verified: true` means the company is registered and active with the tax authority.</Check>
