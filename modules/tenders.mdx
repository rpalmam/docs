The Tenders module gives you programmatic access to public procurement tenders from official government marketplaces. Useful for sales prospecting, competitive intelligence, and compliance monitoring.

## What it returns

- Active and recent tender listings
- Tender details: code, name, status, and closing date
- Awarded contracts: amounts, buyer, and award date
- Search by keyword across tender titles

## Endpoints

### Get company contracts

Returns government contracts where the company is the **provider** (supplier), by tax ID or domain.

```bash
POST /v1/tenders
```

<CodeGroup>

```json Chile
{
  "rut": "77.261.280-K",
  "country": "CL"
}
```

```json Brazil
{
  "rut": "28.591.276/0001-00",
  "country": "BR"
}
```

```json By domain
{
  "domain": "empresa.com.br",
  "country": "BR"
}
```

```json Mexico
{
  "rut": "EMP123456ABC",
  "country": "MX"
}
```

```json Colombia
{
  "identifier": "900123456-1",
  "country": "CO"
}
```

</CodeGroup>

#### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `identifier` | string | тЬЕ* | Tax ID тАФ RUT (CL), CNPJ (BR), RFC (MX), or NIT (CO). Domain resolution not available for MX or CO. |
| `domain` | string | тЬЕ* | Company domain. Resolved to RUT or CNPJ automatically (CL/BR only). |
| `country` | string | тАФ | `CL` (default), `BR`, `MX`, or `CO`. |

*One of `rut` or `domain` is required.

#### Response

```json
{
  "rut": "77.261.280-K",
  "total_orders": 3,
  "orders": [
    {
      "code": "1003473-1501-SE26",
      "name": "Compra equipos",
      "status": "vigente"
    }
  ],
  "source": "mercadopublico"
}
```

### Get awarded contracts (Chile only)

Returns government contracts **won** by the company тАФ adjudicaciones. Includes amounts and buyer details.

```bash
POST /v1/tenders/awarded
```

```json
{
  "rut": "77.261.280-K"
}
```

#### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `rut` | string | тЬЕ* | Chilean company RUT. |
| `domain` | string | тЬЕ* | Company `.cl` domain. |

*One of `rut` or `domain` is required.

#### Response

```json
{
  "rut": "77.261.280-K",
  "total": 2,
  "contracts": [
    {
      "code": "2755-2-LP26",
      "name": "Mantenci├│n sistemas TI",

      "buyer_rut": "61.002.001-3"
    }
  ],
  "source": "mercadopublico"
}
```

### Search tenders

Search active tenders by keyword.

```bash
POST /v1/tenders/search
```

```json
{
  "keyword": "software",
  "country": "CL"
}
```

#### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `country` | string | тЬЕ | `CL`, `BR`, `MX`, or `CO`. |
| `keyword` | string | тАФ | Text to search in tender names. |
| `date_from` | string | тАФ | Filter from date (`DD/MM/YYYY`). |
| `date_to` | string | тАФ | Filter to date (`DD/MM/YYYY`). |

#### Response

```json
{
  "total": 2,
  "results": [
    {
      "code": "3704-10-LS26",
      "name": "Servicio Inform├бtica",
      "status": "cerrada",
      "closes_at": "2026-04-04T18:00:00"
    }
  ],
  "source": "mercadopublico"
}
```

<Note>
  The search endpoint returns up to 20 results per page. Keyword filtering for Brazil (PNCP) is applied client-side on the tender description field.
</Note>
