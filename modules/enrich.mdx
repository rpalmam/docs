---
title: Company Enrichment
description: Get structured firmographic data for Latin American companies by domain or tax ID.
---

The Enrich module returns structured company data from official tax registries. Pass a domain or a tax ID (RUT/CNPJ) and get back a full company profile.

## What it returns

- Legal company name, tax ID, and registration status
- Primary economic activity and CIIU/CNAE code
- Company size classification
- Year of founding
- Tech stack and contact emails (coming soon)

## Supported countries

| Country | Input | Source |
|---------|-------|--------|
| Chile 🇨🇱 | `.cl` domain or RUT | SII |
| Brazil 🇧🇷 | `.com.br` / `.br` domain or CNPJ | Receita Federal via OpenCNPJ |

The country is detected automatically — from the domain TLD or the identifier format. No `country` parameter needed.

## Request

```bash
POST /v1/enrich
```

<CodeGroup>

```json By domain
{
  "domain": "example.cl"
}
```

```json By RUT (Chile)
{
  "identifier": "76.354.771-K"
}
```

```json By CNPJ (Brazil)
{
  "identifier": "47.960.950/0001-21"
}
```

</CodeGroup>

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `domain` | string | conditional | Company domain. Required if `identifier` is not provided. |
| `identifier` | string | conditional | Chilean RUT or Brazilian CNPJ. Skips domain resolution — faster and more accurate when you already have the tax ID. Required if `domain` is not provided. |

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `domain` | string | Queried domain. `null` when request used `identifier` directly. |
| `company_name` | string | Official legal name |
| `identifier` | string | Tax ID — RUT (Chile) or CNPJ (Brazil) |
| `activity` | string | Primary economic activity |
| `activity_code` | string | CIIU (CL) or CNAE (BR) activity code |
| `size` | string | `small`, `large`, or `unknown` |
| `sales_range` | string \| null | Revenue tier from SII tramo (CL only): `micro`, `small`, `medium`, `large`, or `null` |
| `employees_range` | string | Estimated employee range (when available) |
| `region` | string | Administrative region (when available) |
| `founded_year` | integer | Year of first tax registration |
| `status` | string | `active` or `inactive` |
| `tech_stack` | array | Detected technologies (coming soon) |
| `emails` | array | Verified contact emails (coming soon) |
| `country` | string | Detected country code |
| `source` | string | Data source |

## Error responses

| Status | Description |
|--------|-------------|
| `404` | Could not resolve tax ID for the given domain |
| `422` | Country not supported (unsupported TLD) |
| `403` | Missing API key or module not in plan |
| `503` | Data provider temporarily unavailable |
