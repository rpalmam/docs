---
title: Company Enrichment
description: Get structured firmographic data for Latin American companies by domain.
---

The Enrich module returns structured company data from official tax registries. Pass a domain and get back a full company profile — no RUT lookup required.

## What it returns

- Legal company name, RUT, and registration status
- Primary economic activity and CIIU code
- Company size classification
- Year of founding
- Tech stack and contact emails (coming soon)

## Supported countries

| Country | Input | Source |
|---------|-------|--------|
| Chile 🇨🇱 | `.cl` domain | SII via BaseAPI.cl |

## Request

```bash
POST /v1/enrich
```

```json
{
  "domain": "example.cl"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `domain` | string | ✅ | Company domain. Currently supports `.cl` domains. |

## Response

```json
{
  "domain": "example.cl",
  "company_name": "Empresa Ejemplo Spa",
  "rut": "76.354.771-K",
  "activity": "Business Management Consulting Activities",
  "activity_code": "7020",
  "size": "small",
  "employees_range": null,
  "region": null,
  "founded_year": 2018,
  "status": "active",
  "tech_stack": [],
  "emails": [],
  "country": "CL",
  "source": "sii"
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `domain` | string | Queried domain |
| `company_name` | string | Official legal name |
| `rut` | string | Resolved Chilean RUT |
| `activity` | string | Primary economic activity |
| `activity_code` | string | CIIU economic activity code |
| `size` | string | `small`, `large`, or `unknown` |
| `employees_range` | string | Estimated employee range (when available) |
| `region` | string | Administrative region (when available) |
| `founded_year` | integer | Year of first tax registration |
| `status` | string | `active` or `inactive` |
| `tech_stack` | array | Detected technologies (coming soon) |
| `emails` | array | Verified contact emails (coming soon) |
| `country` | string | Detected country code |
| `source` | string | Data source |

## Error responses

| Status | Description |
|--------|-------------|
| `404` | Could not resolve RUT for the given domain |
| `422` | Country not supported (non-.cl domain) |
| `403` | Missing API key or module not in plan |
| `503` | Data provider temporarily unavailable |
