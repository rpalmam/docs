---
title: Company Enrichment
description: Get structured firmographic data for Latin American companies by domain.
---

The Enrich module returns structured company data from official tax registries. Pass a domain and get back a full company profile — no manual ID lookup required.

## What it returns

- Legal company name, tax ID, and registration status
- Primary economic activity and CIIU/CNAE code
- Company size classification
- Year of founding
- Tech stack and contact emails (coming soon)

## Supported countries

| Country | Input | Source |
|---------|-------|--------|
| Chile 🇨🇱 | `.cl` domain | SII via BaseAPI.cl |
| Brazil 🇧🇷 | `.com.br` or `.br` domain | Receita Federal via OpenCNPJ |

The country is detected automatically from the domain TLD. No `country` parameter needed.

## Request

```bash
POST /v1/enrich
```

```json
{
  "domain": "example.cl"
}
```

### Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `domain` | string | ✅ | Company domain. Supports `.cl` (Chile) and `.com.br` / `.br` (Brazil). |

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `domain` | string | Queried domain |
| `company_name` | string | Official legal name |
| `identifier` | string | Tax ID — RUT (Chile) or CNPJ (Brazil) |
| `activity` | string | Primary economic activity |
| `activity_code` | string | CIIU (CL) or CNAE (BR) activity code |
| `size` | string | `small`, `large`, or `unknown` |
| `employees_range` | string | Estimated employee range (when available) |
| `region` | string | Administrative region (when available) |
| `founded_year` | integer | Year of first tax registration |
| `status` | string | `active` or `inactive` |
| `tech_stack` | array | Detected technologies (coming soon) |
| `emails` | array | Verified contact emails (coming soon) |
| `country` | string | Detected country code |
| `source` | string | Data source |

## Error responses

| Status | Description |
|--------|-------------|
| `404` | Could not resolve tax ID for the given domain |
| `422` | Country not supported (unsupported TLD) |
| `403` | Missing API key or module not in plan |
| `503` | Data provider temporarily unavailable |
