---
title: "POST /v1/labor"
description: "Check if a Brazilian company appears on the Lista Suja do Trabalho Escravo (slave labor registry)."
---

<ParamField header="X-API-Key" type="string" required>
  Your API key. Requires the `labor` module.
</ParamField>

## Request body

<ParamField body="identifier" type="string" required>
  Company CNPJ (e.g. `12.345.678/0001-90`). Formatting is optional.
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code. Supported: `BR`.
</ParamField>

## Response

<ResponseField name="identifier" type="string">
  The queried CNPJ, as provided.
</ResponseField>

<ResponseField name="country" type="string">
  Country code.
</ResponseField>

<ResponseField name="has_violations" type="boolean">
  `true` if any records were found in the Lista Suja. `false` if no records found.
</ResponseField>

<ResponseField name="records" type="array">
  List of labor violation records. Empty array if `has_violations` is `false`.
</ResponseField>

<ResponseField name="records[].source" type="string">
  Always `TRABALHO_ESCRAVO`.
</ResponseField>

<ResponseField name="records[].year" type="integer | null">
  Year of the labor inspection finding.
</ResponseField>

<ResponseField name="records[].workers_rescued" type="integer | null">
  Number of workers rescued in this inspection.
</ResponseField>

<ResponseField name="records[].state" type="string | null">
  Brazilian state abbreviation where the violation occurred (e.g. `PA`, `MT`).
</ResponseField>

<ResponseField name="records[].municipality" type="string | null">
  Municipality where the violation occurred.
</ResponseField>

<ResponseField name="records[].activity" type="string | null">
  Economic activity of the employer at the time of inspection.
</ResponseField>

<ResponseField name="records[].decision_id" type="string | null">
  Administrative decision identifier.
</ResponseField>

<ResponseField name="sources_checked" type="array">
  Registries checked in this request.
</ResponseField>

<ResponseField name="checked_at" type="string">
  ISO 8601 datetime of this specific check.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.enrichlatam.com/v1/labor \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"identifier": "12.345.678/0001-90", "country": "BR"}'
```

```python Python
import requests

r = requests.post(
    "https://api.enrichlatam.com/v1/labor",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={"identifier": "12.345.678/0001-90", "country": "BR"},
)
print(r.json())
```
