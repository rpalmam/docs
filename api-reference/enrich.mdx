---
title: "POST /v1/enrich"
description: "Get structured firmographic data for a Latin American company."
---

<ParamField header="X-API-Key" type="string" required>
  Your API key. Requires the `enrich` module.
</ParamField>

## Request body

<ParamField body="domain" type="string" required>
  Company domain (e.g. `example.cl`, `example.com.br`). The API resolves the domain to a tax ID automatically. Supports `.cl` (Chile) and `.com.br` / `.br` (Brazil).
</ParamField>

## Response

<ResponseField name="domain" type="string">
  The queried domain.
</ResponseField>

<ResponseField name="company_name" type="string">
  Official legal company name.
</ResponseField>

<ResponseField name="identifier" type="string">
  Tax ID resolved from the domain — RUT (Chile) or CNPJ (Brazil).
</ResponseField>

<ResponseField name="activity" type="string">
  Primary economic activity description.
</ResponseField>

<ResponseField name="activity_code" type="string">
  Economic activity code — CIIU (Chile) or CNAE (Brazil).
</ResponseField>

<ResponseField name="size" type="string">
  Company size: `small`, `large`, or `unknown`.
</ResponseField>

<ResponseField name="employees_range" type="string">
  Estimated employee range. `null` if unavailable.
</ResponseField>

<ResponseField name="region" type="string">
  Administrative region. `null` if unavailable.
</ResponseField>

<ResponseField name="founded_year" type="integer">
  Year of first tax registration. `null` if unavailable.
</ResponseField>

<ResponseField name="status" type="string">
  Registration status: `active` or `inactive`.
</ResponseField>

<ResponseField name="tech_stack" type="array">
  Detected technologies (coming soon — currently empty).
</ResponseField>

<ResponseField name="emails" type="array">
  Verified contact emails (coming soon — currently empty).
</ResponseField>

<ResponseField name="country" type="string">
  Detected country code.
</ResponseField>

<ResponseField name="source" type="string">
  Data source identifier.
</ResponseField>

<RequestExample>

```bash Chile
curl -X POST https://api.enrichlatam.com/v1/enrich \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain": "example.cl"}'
```

```bash Brazil
curl -X POST https://api.enrichlatam.com/v1/enrich \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domain": "example.com.br"}'
```

```python Python
import requests

r = requests.post(
    "https://api.enrichlatam.com/v1/enrich",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={"domain": "example.cl"},
)
print(r.json())
```
