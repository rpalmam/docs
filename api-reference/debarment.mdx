---
title: "POST /v1/debarment"
description: "Check if a Brazilian company is debarred from government contracts (CEIS + CNEP via Portal da Transparência)."
---

<ParamField header="X-API-Key" type="string" required>
  Your API key. Requires the `debarment` module.
</ParamField>

## Request body

<ParamField body="identifier" type="string" required>
  Company CNPJ (e.g. `12.345.678/0001-90`). Formatting is optional.
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code. Supported: `BR`.
</ParamField>

## Response

<ResponseField name="identifier" type="string">
  The queried CNPJ, as provided.
</ResponseField>

<ResponseField name="country" type="string">
  Country code.
</ResponseField>

<ResponseField name="is_debarred" type="boolean">
  `true` if any records were found in CEIS or CNEP. `false` if no records found.
</ResponseField>

<ResponseField name="records" type="array">
  List of debarment records. Empty array if `is_debarred` is `false`.
</ResponseField>

<ResponseField name="records[].source" type="string">
  Registry that contains this record: `CEIS` or `CNEP`.
</ResponseField>

<ResponseField name="records[].sanction_type" type="string">
  Type of sanction (e.g. `SUSPENSÃO`, `INIDONEIDADE`, `MULTA`).
</ResponseField>

<ResponseField name="records[].sanctioning_body" type="string">
  Body that issued the sanction (e.g. `TCU`, `CGU`, `ANTT`).
</ResponseField>

<ResponseField name="records[].date_start" type="string | null">
  ISO date when the sanction began.
</ResponseField>

<ResponseField name="records[].date_end" type="string | null">
  ISO date when the sanction ends. `null` if indefinite.
</ResponseField>

<ResponseField name="records[].process_id" type="string | null">
  Administrative process number associated with the sanction.
</ResponseField>

<ResponseField name="records[].fine_amount" type="number | null">
  Fine in BRL. Only present in CNEP records — `null` for CEIS.
</ResponseField>

<ResponseField name="sources_checked" type="array">
  Registries checked in this request: `["CEIS", "CNEP"]`.
</ResponseField>

<ResponseField name="checked_at" type="string">
  ISO 8601 datetime of this specific check.
</ResponseField>

```bash cURL

```
