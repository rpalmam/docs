---
title: "POST /v1/kyb"
description: "Verify a company's legal registration status."
---

<ParamField header="X-API-Key" type="string" required>
  Your API key. Requires the `kyb` module.
</ParamField>

## Request body

<ParamField body="identifier" type="string" required>
  Company tax identifier. For Chile: RUT (e.g. `76354771-K` or `763547710`).
</ParamField>

<ParamField body="country" type="string" required>
  ISO 3166-1 alpha-2 country code. Supported: `CL`.
</ParamField>

## Response

<ResponseField name="identifier" type="string">
  The queried tax identifier, as provided.
</ResponseField>

<ResponseField name="country" type="string">
  Country code of the queried company.
</ResponseField>

<ResponseField name="company_name" type="string">
  Official legal company name from the tax registry.
</ResponseField>

<ResponseField name="status" type="string">
  Registration status: `active`, `inactive`, or `not_found`.
</ResponseField>

<ResponseField name="verified" type="boolean">
  `true` if the company has active tax registration.
</ResponseField>

<ResponseField name="founded_year" type="integer">
  Year the company first registered for tax activities. `null` if unavailable.
</ResponseField>

<ResponseField name="end_date" type="string">
  Date of business termination, if applicable. `null` if still active.
</ResponseField>

<ResponseField name="source" type="string">
  Data source identifier. `sii` for Chile.
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST https://api.enrichlatam.com/v1/kyb \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"identifier": "76354771-K", "country": "CL"}'
```

```python Python
import requests

r = requests.post(
    "https://api.enrichlatam.com/v1/kyb",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={"identifier": "76354771-K", "country": "CL"},
)
print(r.json())
```
